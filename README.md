- Como parte de una práctica de clase en el lenguaje PHP, he hecho una aplicación web cuya finalidad es tratar de regular a través de un panel de administrador
  mediante Login las funciones básicar de añadir, activar y desactivar cursos para el profesorada.
- Como usuario podías inscribirte en cursos los cuales en función de ciertas regulaciones te darían unos puntos que junto a un par de requisitos como la previa participación en otros
  cursos definirían quien podría entrar en el curso y quien no, limitado a ciertas plazas determinadas en cada uno de los cursos de forma independiente.
- Respecto a los lenguajes, he utilizado HTML, CSS y PHP
